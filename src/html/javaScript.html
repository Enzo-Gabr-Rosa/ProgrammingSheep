Variáveis.


      
Agora vamos aprender a usar uma variável, mas primeiro, o que é uma váriavel?
        Pense nela como uma caixinha, nessa caixinha vamos colocar um dado(como um número ou texto por exemplo),
        esse dado ficara armazenado na variavel, e sempre que quisermos pegar esse dado, é só chamar a caixinha.
        

        

        Exemplo: var variavel = 1
        

        return variavel;
      


      

      

      

        

      


    

      
    

  



  



    
Contas "variáveis".


    
Variáveis também podem fazer calculos dentro delas para 
      simplificar nossa vida, tente fazer uma conta dentro de uma váriavel que resulte em 15.
      

      

      Exemplo: var variavel = 7 + 42;
      

      return variavel;
